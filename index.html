cite : 웹 문서나 포스트에서 참고 내용을 표시 -> 'cite'
code : 컴퓨터 인식을 위한 소스 코드 입니다. -> function savetheLocal()
small : 부가 정보처럼 작게 표시해도 되는 텍스트입니다. -> 부가정보...
sub : 아래 첨자 -> H2O
sup : 위 첨자 -> E = mc2
s : 취소선 -> 10% 할인가 30,000 27,000
u : 밑줄
ins : 공동 작업 문서에서 새로운 내용을 삽입합니다. -> 새로운 내용
u태그와 차이?
밑줄이 표시되는 것은 같은나
ins : 새로 추가된 내용을 표시, datetime 속성값을 이용, 내용의 추가와 삭제가 언제 이루어졌는지 의미적으로 지정할 수 있다.
datetime 속성은 보이지 않지만 의미적으로 그 시점에 내용의 추가,삭제가 이루여졌음을 나타낸다.
u : 단순히 밑줄만 표시
del : 공동 작업 문서에서 기존 내용을 삭제합니다. -> 기존 내용
s태그와 차이는 위의 ins,u 태그 차이점과 동

ol,li 태그 :
  1. 속성 type='a'(알파벳)
  2. 속성 start='2'(b부터 시작)
  3. ol,li의 순서있는 목록
ul,li 태그 :
  • ul은 순서없는 목록으로 type=''없고
  • 그러므로 start=''도 없다.
  • ul,li의 순서없는 목록

이 들어간다. dt태그 하나에 dd태그를 여러 개 사용할 수 있다.
