테두리 스타일 지정

  • 박스 모델의 방향

    top -> right -> bottom -> left (시계방향)

  • 테두리 스타일을 지정하는 border-style

    • none, 기본값

    • hidden, 감춤, 표 border-collapse: collapse일 때 다른 테두리도 감춤

      -표, border: 0.1rem hidden #000; border-collapse: collapse;
      1행 1열 1행 2열 1행 3열
      2행 1열 2행 2열 2행 3열
      3열

    • solid, 실선

    • dotted, 점선

    • dashed, 짧은 직선

    • double, 이중선, 0.5rem, border-width: 1rem

    • groove, 창에 조각한 것처럼, 홈이 파인 듯 입체 느낌

    • inset

      표
      border-collapse: seperate(기본값)일 경우
      border: 1rem inset #aaa;
      1행 1열 1행 2열 1행 3열
      2행 1열 2행 2열 2행 3열
      3열
      표
      border-collapse: collapse일 경우

      border-collapse: collapse;

      2행 1열 2행 2열 2행 3열
      3열

  • 테두리 두께를 지정하는 border-width

    border-width: 1rem

    border-top-width: 1rem

    border-width: 0.1rem 0.2rem 0.3rem 0.4rem

  • border-color

    border-color: red

  • 둥근 테두리 border-radius

    border-radius: 1.5rem

    border-radius: 50%(또는 너비 크기의 반), 원

    border-top-left-radius: 1.5rem

    border-radius: 1.5rem/1rem(가로 반지름 / 세로 반지름), 타원

    border-top-left-radius: 1.5rem 1rem(가로 반지름 세로 반지름), 특정 꼭지점만 타원
