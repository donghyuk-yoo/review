  • 배치 방법 결정 display

    • block
      인라인 레벨 요소를 블록 요소로
    • inline
      블록 레벨 요소를 인라인 레벨 요소로
    • inline-block
      인라인 레벨 요소와 블록 레벨 요소의 속성을 모두 가지고, 마진과 패딩 지정 가능
      ex) span태그와 같은 인라인요소는 마진이 좌우만 지정가능, 블록화 또는 인라인 블록화로 해결
    • none
      해당 요소를 화면에 표시하지 않음
    • 예제 : 수평 내비게이션

  • 왼쪽이나 오른쪽으로 배치 float

    • left
      문서 왼쪽에 배치

    • right

      오른쪽에 배치
    • none
      기본값
    • display: inline-block과 차이
      수평 내게이션을 만들때 display: inline-block은 기본 마진과 패딩이 있지만, float-left방식은 없어 개별 지정해주어야 하며,
      사용 후에는 clear로 플로팅을 해제하는 과정이 필요하다.
    • 예제
      p태그는 블록요소
      img태그는 인라인요소, float: left로 띄움

      display 속성은 요소의 종류를 변경하는 것으로 float 속성과 별개, 띄우는 순간 블록,인라인의 의미가 없어진다.

      margin속성으로 p태그 요소와 여백확보
