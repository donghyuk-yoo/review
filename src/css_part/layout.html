웹 문서 레이아웃

  • 배치 방법 결정 display

    • block
      인라인 레벨 요소를 블록 요소로
    • inline
      블록 레벨 요소를 인라인 레벨 요소로
    • inline-block
      인라인 레벨 요소와 블록 레벨 요소의 속성을 모두 가지고, 마진과 패딩 지정 가능
      ex) span태그와 같은 인라인요소는 패딩이 좌우만 지정가능, 블록화 또는 인라인 블록화로 해결
    • none
      해당 요소를 화면에 표시하지 않음
    • 예제 : 수평 내비게이션
