- 배경색, 배경 범위

  • 배경색 지정 background-color
    background-color: #008000;
    background-color: rgb(0,128,0);
    background-color: green;

  • 배경색의 적용 범위 조절 background-clip

    종류 설명
    border-box 박스 모델의 가장 외곽인 테두리까지 적용. 기본값
    padding-box 박스 모델에서 테두리를 뺀 패딩 범위까지 적용
    content-box 박스 모델에서 내용(콘텐츠) 부분에만 적용
    border-box
    padding-box
    content-box

- 배경 이미지

  • 웹 요소에 배경 이미지를 넣는 background-image
    background-image: url('이미지 파일 경로')
  • 배경 이미지의 반복 방법 지정 background-repeat
    종류 설명
    repeat 브라우저 화면에 가득 찰 때까지 가로와 세로 반복, 기본값
    repeat-x 브라우저 화면 너비에 가득 찰 때까지 가로로 반복
    repeat-y 브라우저 화면 높이에 가득 찰떄까지 세로로 반복
    no-repeat 한 번만 표시하고 반복하지 않음
    • repeat
    • repeat-x
    • repeat-y
    • no-repeat
  • 배경 이미지 위치 조절 background-position: <수평위치> <수직위치>
    • background-position: right bottom; (키워드)
      영역크기 w:30rem h:30rem
    • background-position: 50% 50%; (백분율)
      영역크기 w:30rem h:30rem
    • background-position: 21rem 20rem; (크기)
      영역크기 w:30rem h:30rem

  • 배경 이미지의 적용 범위 조절 background-origin

    종류 설명
    content-box 박스 모델에서 내용 부분에만 배경 이미지 표시, 기본값
    padding-box 박스 모델에서 패딩까지
    border-box 박스 모델에서 테두리까지

  • 배경 이미지 고정 background-attachment

    종류 설명
    scroll 화면 스크롤시 배경 이미지도 스크롤, 기본값
    fixed 화면 스크롤시 배경 이미지 고정, 내용만 스크롤

    • scroll

    • fixed

  • background 하나의 속성으로

    background: url('...') no-repeat center bottom fixed;

  • 배경 이미지 크기 조절 background-size

    종류 설명
    auto 원래 배경 이미지 크기만큼 표시, 기본값
    contain 요소 안에 배경 이미지가 다 들어오도록 이미지를 확대, 축소
    cover 배경 이미지로 요소를 모두 덮도록 이미지를 확대, 축소
    크기 이미지의 너비와 높이를 지정, 값이 하나만 주어질 경우 너빗값으로 인식,
    이미지의 너비와 너빗값에 맞춘 높잇값도 자동 계산
    백분율 배경 이미지가 들어갈 요소의 크기를 기준으로 값을 백분율로 지정하고
    그 크기에 맞도록 배경 이미지를 확대, 축소

    • auto

    • contain

    • cover

    • 크기, 가로,세로 30rem, 테두리 영역: 가로,세로 30rem

    • 백분율, 가로 100%, 세로 100%

    • 예제

  • 그러데이션

    • 선형 그러데이션 linear-gradient(to 방향 또는 속도, 색상 중지점, 색상 중지점...)

      방향
      background: linear-gradient(to bottom, white, yellow);
      각도
      background: linear-gradient(45deg, white, yellow);

    • 색상 중지점(색상이 바뀌는 지점)

      linear-gradient(to bottom, #06f, #fff 30%, #06f);

    • 원형 그러데이션 radial-gradient(모양 크기 at 위치, 색상 중지점, 색상 중지점...)

      모양 미지정시 타원형(ellipse)
      background: red; background: radial-gradient(white, yellow, red);
      모양 지정 circle
      background: red; background: radial-gradient(circle, white, yellow, red);
