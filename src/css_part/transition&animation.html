    transform: translate(2rem, 0);

    transform: rotate(45deg);

트랜지션

웹 요소의 스타일 속성이 바뀌는 것

  • 트랜지션의 대상 지정 transition-property

    • all(기본값, 모든속성에 적용)

    • none

    • 속성이름(transition-property: width, height;)

  • 트랜지션의 진행 시간 지정 transition-duration

    transition-property: width, height;
    transition-duration: 2s, 1s; (width에 2초, height에 1초)

    hover { width: 20rem; height: 20rem; }

  • 트랜지션의 속도 곡선 지정 transition-timing-function

    activate all
    ease
    lienar
    ease-in
    ease-out
    ease-in-out

  • 트랜지션의 지연 시간 설정 transition-delay

    transition-delay: 2s;

    hover { width: 20rem; height: 20rem; }

  • 트랜지션 속성 한번에 transition

    transition: all 2s ease-in 0.5s;
